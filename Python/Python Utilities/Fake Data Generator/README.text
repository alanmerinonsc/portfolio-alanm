Data Faker – Synthetic E-Commerce Dataset Generator
Data Faker is a Python script that generates realistic, synthetic datasets for a fictional e-commerce platform. 
It uses the Faker library to simulate customer profiles, product listings, order histories, 
and payment records, then exports the data into structured CSV files.

The script creates five datasets:
• customers.csv: Contains 100 fake customers with names, unique emails, and account creation dates spanning the past two years.
• products.csv: Includes 100 products with SKU codes, randomized names, and prices ranging from $10 to $500.
• orders.csv: Generates 200 orders linked to customers, each with a total amount, order date within the past year, and a status (pending, completed, or cancelled).
• order_items.csv: Adds 500 line items that connect products to orders, specifying quantity and unit price.
• payments.csv: Records one payment per order, with payment date, amount, and status (paid, failed, or refunded).

All timestamps are generated to reflect realistic business activity. 
The script uses Python’s built-in csv module and the Faker library. 
To run it, install Faker using pip install faker, then execute the script with Python 3. 
The output files will be saved in the current directory, and a confirmation message will appear once generation is complete.

Data Faker is ideal for testing data pipelines, practicing analysis with pandas, simulating database imports, or building mock dashboards and APIs.
