## C# Developer Utilities

A set of **command-line developer tools written in C#** designed to automate integration tasks and authentication workflows.

### Example Project

**Salesforce JWT Authentication CLI**

A utility that automates the Salesforce OAuth JWT Bearer Flow.

#### Features

- RSA key pair and X.509 certificate generation  
- RS256 signed JWT assertion creation  
- OAuth token exchange with Salesforce APIs  
- Support for both production and sandbox environments  

This tool simplifies a process that typically requires multiple manual cryptographic and authentication steps.
